## deskripsi

## iterative statement adalah inti dari logika percabangan. Program akan mengevaluasi suatu kondisi (benar atau salah) sebelum memutuskan langkah mana yang diambil. Ibaratnya seperti menghadapi persimpangan jalan: Anda memilih jalur A atau jalur B berdasarkan petunjuk jalan. tujuannya untuk membuat keputusan
## Iterative statement digunakan untuk menjalankan blok kode yang sama berulang-ulang selama kondisi tertentu masih terpenuhi. tujuannya untuk Melakukan efisiensi pada tugas yang berulang.
## Transfer statement berfungsi untuk mengalihkan kendali program dari satu bagian ke bagian lain secara tiba-tiba. Ini biasanya digunakan di dalam Conditional atau Iterative untuk menghentikan proses atau melompati langkah tertentu. Tujuannya untuk Interupsi atau mengubah alur eksekusi normal.
